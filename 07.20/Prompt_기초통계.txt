# Outlier 처리

df_train 데이터에서 연속형변수의 경우 price 를 제외한 변수들을 |Z| > 3 기준으로 이상치 여부를 판단하고, 
이상치로 판단되는 값은 경계값으로 df_train과 df_test 를 대체하는 코드 보여줘
 

# Log 변환

df_train 데이터에서 연속형 변수 중 price를 제외한 변수들을 log 변환하고,
변환된 변수 + price 변수 + 범주형 변수를 모두 병합하는 코드를 보여줘.
단, log 변환 시 음수값이 존재할 수 있으므로 signed log transformation 으로 처리해줘.
그리고 df_test 데이터에서도 동일한 변환 기준을 적용하고
변환된 변수 + price 변수 + 범주형 변수를 모두 병합해줘.
결과 데이터는 동일한 이름(df_train, df_test)으로 유지해줘.


# 데이터 Scaling

df_train 데이터에서 연속형변수의 경우 price 를 제외한 변수들은 Min-Max Scaling 으로 표준화하고, 
표준화된 변수 + price 변수 + 범주형 변수를 모두 병합하는 코드 보여줘.
그리고 df_test 데이터에서도 동일한 표준화 기준을 적용하고 표준화된 변수 + price 변수 + 범주형 변수를 모두 병합해줘.
결과 데이터는 동일한 이름으로 해줘
